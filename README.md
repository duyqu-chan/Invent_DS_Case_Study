# Invent DS Case Study: Retail Promotion Bump Analysis

This project is a data science case study developed to analyze and understand the impact of promotions on sales (promotion bump) in the retail sector.

## Project Summary
The project focuses on analyzing sales increases during promotional periods and identifying which factors are most effective in driving these increases. It includes data cleaning, exploratory data analysis (EDA), and visualization steps.

## File Structure
* notebooks/01_promotion_bump_analysis.ipynb: Main analysis file containing data processing and visualizations.
* data/: Raw and processed datasets used for the analysis.
* docs/: Reports and presentation.
* outputs/: Charts, figures and results generated from the analysis.
* requirements.txt: Necessary Python libraries to run the project.

## Technologies and Libraries
The following Python data science tools were utilized in this project:
* Pandas and NumPy: Data manipulation and numerical calculations.
* Matplotlib and Seaborn: Data visualization.
* Jupyter Notebook: Interactive analysis environment.

## Installation and Execution
To run this project in your local environment:

1. Clone this repository:
   git clone https://github.com/duyqu-chan/Invent_DS_Case_Study.git

2. Install the required libraries:
   pip install -r requirements.txt

3. Launch Jupyter Notebook and open the analysis file.
